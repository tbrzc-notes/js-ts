- switch en la expresion y en cada case utiliza el operador estrictamente igual [[===]]
```js
switch (color) {//expresion
  case "rojo":
  case "azul":
  case "verde":
    console.log("Color primario");
    break;
  default:
    console.log("Otro color");
}
```

## simular if-else
```js
switch (true) {
  case nota === 10 && hizoTp:
    // *promociona*
    break;
  case nota > 4:
    // *aprobado*
    break;
  case !hizoTp:
    // *recursa*
    break;
  default:
    // *recupera*
}
```

## concatenacion
```php
$mensaje = "Hola ";
$nombre = "Javier";
$mensaje .= $nombre;
$saludo = $mensaje . ", bienvenido!";
```

## interpolacion
```php
$nombre = "Javier";
echo "Hola {$nombre}, bienvenido!";
```

## sprintf()

```php
$nombre = "Javier";
$edad = 30;
echo sprintf("Nombre: %s, Edad: %d", $nombre, $edad);
```
